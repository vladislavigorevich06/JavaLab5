## **Задание 1**

**Задача 1**
Создать дробь, которая умеет кэшировать вычисление вещественного значения.


  * `getDoubleValue()` — возвращает вещественное значение дроби (использует кэш)
  * `setNumerator(int)` / `setDenominator(int)` — изменяют числитель и знаменатель, сбрасывают кэш
  * `toString()` — строковое представление `"числитель/знаменатель"`

**Алгоритм решения:**

1. Создан интерфейс с методами для работы с дробью.
2. Реализован базовый класс дроби с проверкой знаменателя и корректной обработкой отрицательных значений.
3. Создан кэш для хранения вычисленного значения. При изменении дроби кэш сбрасывается.

* Метод `processPeopleFile(String filename)`

4. Группируем имена по номеру.
